# DjangoFormInfo
基于django的易班在线填表系统

# 项目进展

## 已完成：

1. 管理员使用excel表格进行创建问卷，可选择行优先和列优先
2. 用户填写表格，答卷信息同步到数据库，填写完成后表格不会在列表展示
3. 管理员在数据中心直接下载所有问卷信息
4. 仅支持易班登录，和易班相结合
5. 默认数据库更改为mysql适合高并发访问

## 已知BUG

1. 问卷填写后根据url可重复提交，数据库会奔溃
2. 用户模型有问题，用户名按说允许重复
3. 调用不了真实姓名api
